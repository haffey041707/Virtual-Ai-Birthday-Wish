# Project 1 - Rule-Based AI Chatbot

## Goal
Build a simple rule-based chatbot that responds to predefined user inputs.

## Requirements Covered
- Handles greetings (`hi`, `hello`, `hey`, etc.)
- Handles exit commands (`bye`, `exit`, `quit`, etc.)
- Uses `if/elif/else` decision logic for responses
- Runs continuously until the user exits the chat

## Files
- `index.html` - Chatbox UI
- `styles.css` - Styling for the chatbox
- `script.js` - Rule-based chatbot logic in the browser
- `chatbot.py` - Optional terminal version using `while True`

## Run (Browser Chatbox)
1. Open `index.html` in your browser.
2. Start chatting in the input box.
3. Use `bye`, `exit`, or `quit` to end the chat.

## Run (Terminal Version)
```bash
python3 chatbot.py
```

## Example Inputs
- `hi`
- `what is your name`
- `how are you`
- `help`
- `bye`
